Detailed Description

Contains image storage and processing classes for the MLX90640 camera driver.

This module provides classes for storing and processing images captured by the MLX90640 thermal infrared camera driver.

RAW VERSION This version is a stripped-down MLX90640 driver that produces only raw data, not calibrated data, in order to save memory.
