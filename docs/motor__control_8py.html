Detailed Description

Defines a MotorDriver class for controlling motors in an ME405 kit.

This script defines a MotorDriver class, which allows controlling motors by initializing GPIO pins and setting the duty cycle to control torque.

Note
This class assumes that the motor driver is used in an ME405 kit.
